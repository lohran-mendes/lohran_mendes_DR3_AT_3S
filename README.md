# InfnetFood

Aplicativo de delivery de comida desenvolvido com React Native e Expo feito para o Assessement da matéria de Desenvolvimento Mobile com React Native.

## Como rodar

```bash
# instale as dependências
npm install

# rode o projeto
npm run start
```

Apos iniciar, escaneie o QR code com o Expo Go ou abra em um emulador.

Eu tive problemas para rodar o projeto dentro do meu celular, então criei o seguinte script para rodar o projeto usando o --tunnel, ele cai e para de funcionar com frequencia, mas é possivel testar ele rodando novamente o comando quando acontece

```bash
# rode o projeto usando o tunnel
npm run start:phone
```
